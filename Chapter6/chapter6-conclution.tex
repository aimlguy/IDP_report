\section{Conclusion}
This research successfully developed and tested an AI-driven Digital Twin for monitoring and optimizing industrial energy consumption in MSMEs. Using daily energy logs from a physically constrained machining facility, the mathematical application of quadratic regression proved highly effective, achieving a prediction accuracy of $R^2 = 0.965$. This accuracy enables the twin to reliably model potential operational constraints without requiring physical implementation on the factory floor. Simulation scenarios demonstrated clear financial potential: shifting the load profile to eliminate diesel dependency entirely generates a 60\% reduction in energy expenditure alongside an 836 kg drop in daily CO\textsubscript{2} emissions. Ultimately, the system establishes a scalable and affordable framework that integrates data analytics effectively into traditional manufacturing environments.

\section{Future Work}
\begin{itemize}
    \item \textbf{High-Frequency Edge Deployment:} Evolving from daily historical records to real-time integration utilizing Wi-Fi enabled Edge IoT sensors measuring running parameters at sub-second intervals.
    \item \textbf{Automated Alert Processing:} Deploying script algorithms to trigger maintenance alerts natively when instantaneous machine load deviates drastically from established prediction curves.
    \item \textbf{Neural Network Integration:} Implementing robust Long Short-Term Memory (LSTM) topologies to map complex seasonal weather variants and ambient thermal effects on heavy inductive equipment.
\end{itemize}

\section{Learning Outcomes}
\begin{itemize}
    \item Demonstrated proficiency in extracting predictive logic and Key Performance Indicators directly from raw operational facility data.
    \item Successfully built and validated foundational regression models evaluating non-linear thermal scaling boundaries of industrial machineries.
    \item Quantified abstract sustainable concepts (EPU tracking and Emissions footprints) translating theoretical equations into precise real-world monetary returns.
\end{itemize}
