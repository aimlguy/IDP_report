\section{MATLAB Regression Snippets}
The following MATLAB commands extract coefficients utilizing robust fitting models:
\begin{tcblisting}{listing only,colback=gray!10!white, breakable, boxsep=0pt,top=1mm,bottom=1mm,left=1mm,right=1mm,
listing options={
language= Matlab,
basicstyle=\small\ttfamily, 
commentstyle=\color{gray},
keywordstyle=\color{blue}
}}
% Load historical MSME Dataset
data = readtable('energy_dataset.xlsx');

% Preprocessing
production = data.Production;
totalEnergy = data.E_eb + data.E_dg;

% Polynomial Fitting (Quadratic)
p = polyfit(production, totalEnergy, 2);

% Predicting values based on the quadratic curve
predictedEnergy = polyval(p, production);

% Calculate Correlation Coefficient
R_sq = 1 - sum((totalEnergy - predictedEnergy).^2) / sum((totalEnergy - mean(totalEnergy)).^2);
disp(['R-squared value: ', num2str(R_sq)]);
\end{tcblisting}
